# Curlist

The very experimental tool for creating curated list. 
The mission is to create a framework for creating data oriented web applications.

## Requirements

- Node >= 8.x
- SQLite or PostgreSQL
- Elasticsearch
- Redis

## Getting started

In progress..

## Current features

- exploring data by filters and full text search
- items changelog
- items editing 
- fast view
- list of all filters
